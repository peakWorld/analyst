# 分析项目

## 项目开发
* 执行`npm run build:watch`, 开启文件监听
* 执行`npm start`, 启动项目

## 链接命令行
* 执行`sudo npm link`

## 命令行执行输出开发日志
* `DEBUG=* sable assets -x 'acb.json'`
